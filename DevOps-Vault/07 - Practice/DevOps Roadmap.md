## What to Learn Next

| Phase | Topic | Timeline | Key Tools |
|-------|-------|----------|-----------|
| **2** | Containerization | 2-4 weeks | Docker deep dive, security, registries |
| **3** | Orchestration | 4-6 weeks | Kubernetes, Pods, Services, Helm |
| **4** | CI/CD | 2-3 weeks | GitHub Actions, Jenkins, GitLab CI, ArgoCD |
| **5** | Cloud | 4-6 weeks | AWS/GCP/Azure, EC2, S3, RDS, VPC, IAM |
| **6** | IaC | 3-4 weeks | Terraform advanced, Ansible, Packer |
| **7** | Monitoring | 2-3 weeks | Prometheus, Grafana, ELK, SLOs |
| **8** | Advanced | Ongoing | Service meshes, GitOps, DevSecOps, Platform Engineering |

> [!devops] **Total time to job-ready DevOps Engineer: 6-12 months**
